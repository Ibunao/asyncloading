
实现过程
1、异步读取网络中json数据
2、创建适配器，将json中的数据进行绑定
3、对图片进行多线程（异步）处理，解决错位问题
4、缓存优化
5、监听优化（滚动时不更新UI，停止时更新）

过程详解
1、异步读取网络中json数据
    创建一个储存所需json对应数据的对象，用list集合进行存储
    异步读取json的值存入list集合中，并绑定适配器，将数据传递过去
2、创建适配器，将json中的数据进行绑定
    创建适配器继承与BaseAdapter，进行数据的绑定
3、对图片进行多线程（异步）处理，解决错位问题
    利用了两种多线程的方法Thread和AsyncTask两种方法
    解决错位问题，在适配器中为每一个ImageView标签通过setTag
    在（中线程中）异步加载图片进行输出的时候进行判断，输出，这样就可以了
4、缓存优化
    创建缓存LruCache，将加载的图片添加到缓存
5、监听优化（滚动时不更新UI，停止时更新）
    实现AbsListView.OnScrollListener接口
